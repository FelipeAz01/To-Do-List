# ✅ To-Do List
![Status - Em Desenvolvimento](https://img.shields.io/badge/status-em_desenvolvimento-yellow)
![Badge Versão](https://img.shields.io/badge/versão-1.0.0-blue)

Este projeto é uma lista de tarefas interativa onde o usuário pode adicionar, completar ou remover tarefas de forma simples e intuitiva. É ideal para organizar e acompanhar suas atividades diárias.

## 📘 Descrição
A To-Do List foi desenvolvida para ajudar o usuário a gerenciar suas tarefas de forma prática. O design limpo e interativo torna fácil adicionar, marcar como concluída ou excluir tarefas. O destaque visual das interações (como riscar e colorir tarefas concluídas) melhora a experiência do usuário.

## 🖼️ Características
## 🖋️ Adicionar Tarefas
- Campo de entrada com a mensagem: "O que tenho que fazer...".
- Botão Adicionar que insere a tarefa na lista.
  
## 📋 Gerenciar Tarefas
- [x] As tarefas adicionadas aparecem em uma lista abaixo do campo de entrada.
- [x] Quando o usuário passa o mouse sobre uma tarefa, dois ícones aparecem:
- [x] Check Verde (à esquerda): <br>
      - Quando clicado, a tarefa é riscada e sua cor muda para verde, indicando que foi concluída.
- [x] Lixeira Vermelha (à direita):<br>
      -Remove a tarefa da lista.
## 💻 Interface Responsiva
Layout otimizado para dispositivos móveis e desktops, garantindo uma experiência fluida em qualquer tela.


## 🛠️ Tecnologias Utilizadas

- HTML: 🌐📝 Para estruturar a interface, incluindo o campo de entrada, botão e lista de tarefas.
- CSS: 🎨📐 <br>
        - Para estilizar os elementos e criar os efeitos visuais (como cores e riscar tarefas concluídas).<br>
        - Responsividade para diferentes tamanhos de tela.

- JavaScript: 💻✨ <br>
        - Adicionar tarefas dinamicamente à lista. <br>
        - Implementar o comportamento dos ícones de check e lixeira.<br>
        - Manter interatividade com eventos de clique e hover.
